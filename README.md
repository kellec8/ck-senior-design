# ck-senior-design
Senior Design Project at University of Cincinnati

## Team: CK

Chase Keller (CS, kellec8@mail.uc.edu)

Project Topic Area

Web Development

## Project Abstract

A website that will host a game made in React about cooking food under a certain amount of time.  Will involve database elements, but much of the focus will be on the design of the frontend.

## Project Description

This game will have the player attempt to cook food under a certain amount of time.  The challenge will come from the player needing to cook faster or more difficult foods over time.

The game will be developed using React, Postgres and Express.

## Budget

To date, there have been no expenditures for this project.
